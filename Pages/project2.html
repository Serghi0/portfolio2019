Project 2:

The SpaceX Clock

Project Description

For the school subject, IxD & Technology II, the objective was to create a website for passengers on a SpaceX spacecraft where they could see the time on earth wherever they are. The tasks were to indicate on the screen if it was day time or night time and to show the time in one or more time zones. The target was also to use the TweenMax library to create animations on the website. The concept of the design was a rocket flying from earth to mars where the "space" would indicate what time of the day it is. It will be dark when it's at night, it will be sky blue if it is day time and it would be orange if it is dusk or dawn. Furthermore, the time of the clock determines the background color. You can also choose different time-zones to see on the clock.

This was a solo project where I worked solely on the designs and front-end development of the website. I had to do research beforehand on the different time-zones differences and how to apply them to code. I also had to do research on how to apply the TweenMax library to the website.

Design and front-end development of the website

This website's designs were created using Lo-Fi prototypes where sketches were made to visualize how the page would look and which elements needed to be added. Furthermore, Hi-Fi prototypes were created to visualize the animations and interactions of the website. For the development, HTML was used to apply the content to the website. Additionally, CSS was applied to the website to create the layout of the website. The layout was created mobile first where media queries were used. The layout was created using CSS grid layout and CSS Flexbox. Furthermore, the TweenMax library was used to create the animations in JavaScript. The clock operations were also created in JavaScript.

Highlighted elements

What I'm most proud of this project, is the TweenMax animations that has been applied to the website. The animations were applied to different elements. Furthermore, the animations play in the chosen order to highlight to the user that they are "on a journey to mars". I am proud of this achievement because learning how to apply the TweenMax library took some time to learn. Furthermore, the animations really bring life to the website. I am also proud of the clock that I created for the website. I was able to apply different time-zones to the clock so that people from different parts of the world could use this website. I had to use JavaScripts techniques which I normally have no experience with, and I applied these successfully. Now I can adept these codes for future projects, which I am happy for.
